# Course Peer Observation — Computer Science Foundation

A bilingual (English / 繁體中文) web application for the Course Peer Observation program, built with Next.js and deployed on Vercel.

**Live site:** https://emi-cs.aiplab.tw

## Features

- Bilingual support (English primary, Traditional Chinese)
- Peer observation schedule and process information
- Observation feedback form
- Teaching resources for EMI and CS pedagogy
- Responsive design with Tailwind CSS

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site locally.

## Deployment

Deployed on [Vercel](https://vercel.com) with custom domain `emi-cs.aiplab.tw`.
